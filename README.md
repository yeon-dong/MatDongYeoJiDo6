## 맛집 탐방의 대가 "맛동연"이 여러분께 서울의 맛집을 소개해드립니다!

맛동연 경력
- 1999년 맛의 고장 전라도 출생
- 2000년 전라도 출생 답게 반찬 3종 모두 섭취
- 2009년 초등학생 전교생 대상 이쁜 이빨 대회 1위
- 2015년 전라도 50년 이상 거주하신 친할머니에게 지금까지 본 사람 중 제일 잘 먹는다 인정
- 2019년 군입대
- 2020년 맛동연 eat_yeon_ 계정 운영
